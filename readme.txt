# ShoppyGlobe Backend API

This repository contains the backend API for the ShoppyGlobe e-commerce application, built with Node.js, Express, MongoDB, and JWT authentication.

## Running the API

1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Create a `.env` file based on the provided example.
4. Start MongoDB locally.
5. Run `npm run dev` to start the server.

## API Documentation

See [API.md](API.md) for detailed information on all API endpoints, testing instructions, and screenshots.

## Testing

All API endpoints have been tested using ThunderClient. See the `/screenshots` folder for testing screenshots.
